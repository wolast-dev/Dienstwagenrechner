Wichtigste Tipps

GVV (Geldwerter Vorteil): Listenpreis x Satz (0,25% Elektro / 0,5% Hybrid / 1% Verbrenner). Das Finanzamt rechnet das wie Lohn.

Leasingrate: Zahlt der AG -- Sie zahlen nur die Mehrsteuer auf den GVV.

Turboabschreibung 75%: AG kann E-Auto im Jahr 1 mit 75% abschreiben (Jul 2025-Dez 2027). Starkes Verhandlungsargument!

Leasingfaktor: Rate / Listenpreis x 100. Unter 1% = gut. Ueber 1,3% = verhandeln.

Homeoffice: Einzelbewertung statt Pauschale kann GVV deutlich senken.
